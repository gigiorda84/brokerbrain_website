# Broker-Brain — Sito Web

Sito web ufficiale di **Broker-Brain**, piattaforma AI per la mediazione creditizia.

## Struttura

```
broker-brain-site/
├── index.html      # Sito completo (HTML + CSS + JS)
├── img/            # Immagini generate
└── README.md
```

## Sezioni

- **B2C (Clienti)** — Finanziamenti a rate per privati: dentale, auto, ottica, veterinario, salute, casa
- **B2B (Partner)** — Dashboard e integrazione per rivenditori auto, dentisti, ottici, veterinari e altri partner
- **Team** — Il team di Broker-Brain
- **Contatti** — Form con selezione profilo (cliente / azienda / rivenditore / dentista)

## Deploy

Il sito è un singolo file HTML con CSS e JS integrati. Per pubblicarlo:

### GitHub Pages
1. Vai su Settings > Pages
2. Seleziona il branch `main` e la cartella `/ (root)`
3. Il sito sarà live su `https://tuousername.github.io/broker-brain-site/`

### Netlify / Vercel
Collega il repository e il deploy è automatico.

## Tecnologie

- HTML5 / CSS3 / JavaScript vanilla
- Font: Inter (Google Fonts)
- Immagini generate con Python PIL
- Nessuna dipendenza esterna
